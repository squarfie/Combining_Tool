# How to Use

1. Run the executable

Navigate to the dist folder and double-click the .exe file (e.g., Combining_Tool.exe).

2. Provide folder paths

When prompted, enter:

-- The input folder path – make sure both Excel files are inside this folder.

-- The output folder path – this is where the combined Excel file will be saved.

3. Enter file names

Input the names of the two Excel files you wish to merge (e.g., file1.xlsx, file2.xlsx).

4. Check output

Once merged, the combined file will be automatically saved in the specified output folder.

5. Repeat or exit

After each merge, you will be asked whether to combine another set of files:

-- Enter y to continue with another combination.

-- Enter n to exit the application.

# System Requirements

Operating System: Windows

No installation required – the executable contains all necessary dependencies.

# Building the Executable (Optional – For Developers)

If you want to create an executable file from the Python script:

Open Command Prompt and navigate to the folder where your script is located:

ex: D:\PROGRAMMING\Combining_Tool> pyinstaller --onefile Combining_Tool.py

This will create a standalone .exe in the dist folder.

Share in any computer the .exe file from the dist folder.

It can be run on any Windows machine without Python or additional setup.
